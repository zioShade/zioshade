# Trademark Policy

zioshade is an independent, open-source project: a pure-Zig compiler that
translates GLSL into SPIR-V and cross-compiles SPIR-V into HLSL, MSL, GLSL, and
WGSL.

## Our use of third-party names

zioshade targets and interoperates with several shading languages, intermediate
formats, and reference tools that are named or trademarked by their respective
owners, including (non-exhaustively):

- **SPIR-V**, **Vulkan**, **GLSL**, and **OpenGL** -- Khronos Group.
- **HLSL** and **DirectX** -- Microsoft.
- **MSL** (Metal Shading Language) and **Metal** -- Apple.
- **WGSL** and **WebGPU** -- W3C.
- **glslang** and **SPIRV-Cross** -- Khronos Group (the reference toolchain
  zioshade is differentially validated against).

We use these names **nominatively** -- that is, only to describe which language,
format, or reference tool zioshade reads, writes, or is compared to. This is the
same descriptive use that lets a product state it is "compatible with" or "for"
another technology. Without these names it would be impossible to describe what
zioshade does.

## We are not affiliated

- zioshade is **not affiliated with, endorsed by, or sponsored by** Khronos Group,

- zioshade does **not** use third-party logos, wordmarks, or other branded trade
  dress. Only plain-text names, and only as much of a name as is necessary to
  identify the language, format, or tool.
- References to **glslang** and **SPIRV-Cross** describe an independent reference
  implementation used to differentially verify zioshade's output. zioshade contains

## Renaming on request

We respect trademark holders' brands. If you hold a trademark and believe a use in
this project is confusing or infringing, please open an issue or contact the
maintainer; we will address it promptly.
