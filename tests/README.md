# Test Shader Attribution

The test shaders in this directory are sourced from third-party projects.
Each subdirectory contains its own `LICENSE` file with the full license text.

## tests/glslang-430/

Source: [KhronosGroup/glslang](https://github.com/KhronosGroup/glslang) — `Test/` directory
License: **BSD 3-Clause License**
Files: 41 GLSL test shaders (`.frag`, `.vert`, `.comp`)

## tests/spirv-cross/

Source: [KhronosGroup/SPIRV-Cross](https://github.com/KhronosGroup/SPIRV-Cross) — `shaders/` and `reference/` directories
License: **Apache License 2.0**
Files: 302 files — 264 GLSL source shaders + 89 SPIR-V assembly reference files (`.asm.*`)

## tests/ghostty/

Source: [deblasis/wintty](https://github.com/deblasis/wintty) — `src/renderer/shaders/`
License: Project's own shaders.
Files: 10 GLSL renderer shaders
